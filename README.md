# Grape on Rack Starter Kit

## Ruby Version
```
2.4.0
```

## Features

- Grape on Rack
- ActiveRecord
- Postgresql
- Puma
- Active Record stand alone migration
- paranoia soft deleted
- jwt
- bcrypt
- mina deployment
- rspec and factory_girl

## Setup

1. clone this repo
`git clone https://github.com/Edison-Hsu/grape-on-rack-starter`

2. remove .git folder
```
cd REPO_FOLDER
rm -rf .git/
```
